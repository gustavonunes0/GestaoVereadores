Task 14 — Base de Comissões
Módulo
src/legislativo/comissoes
Objetivo

Implementar comissões sem exigir parecer formal no MVP.

Regras
Comissão pertence ao tenant.
Comissão pode ser permanente ou temporária.
Comissão possui finalidade.
Comissão possui período opcional.
Comissão possui status.
Parecer formal fica para task futura.
Critério de aceite
Admin/staff consegue cadastrar comissão e definir tipo/finalidade.
